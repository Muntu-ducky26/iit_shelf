# iit_shelf_test

A web-first IIT Shelf repository with:

- PHP backend APIs in `backend/`
- React/Vite frontend in `web frontend/`

## Getting Started

Install dependencies:

```bash
npm install
```

Run the backend and web frontend together:

```bash
npm run dev
```

Frontend:
- `http://127.0.0.1:5173`

Backend:
- `http://127.0.0.1:8000`
